# Local Kart Racer

A locally runnable arcade kart racing game built with `HTML`, `CSS`, `JavaScript`, `Vite`, and `Three.js`.

## Run Locally

```bash
npm install
npm start
```

Open the local URL printed by Vite in your browser.

## Controls

- `W` / `Up Arrow`: accelerate
- `S` / `Down Arrow`: brake / reverse
- `A` / `D` or `Left` / `Right`: steer
- `Shift`: drift
- `Space`: use held item
- `R`: reset kart to track
- `Esc`: return to track select

## Features

- Two playable tracks
- AI racers
- Drift and boost handling
- Item boxes and pickups
- Track selection menu

## Tech Stack

- `Vite`
- `Three.js`
- Vanilla `HTML`, `CSS`, and `JavaScript`
